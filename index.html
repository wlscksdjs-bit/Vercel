완벽한 CLAUDE 마스터리 로드맵

초보자에서 파워 유저가 되기까지, 당신이 가야 할 유일한 로드맵

메인 로드맵 제1장. 기초 이해
10단계 마스터리 로드맵 (클릭하여 상세내용 학습)

완벽한 프롬프트 공식

  • 역할 (ROLE) 역할 지정
  • 맥락 (CONTEXT) 배경, 전문, 상황 설명
  • 과제 (TASK) Claude에게 원하는 작업
  • 제약 (CONSTRAINTS) 규칙, 톤, 길이, 스타일
  • 형식 (FORMAT) 출력 형식 지정
예시 프롬프트
역할:[전문가 역할] 맥락:[배경/상황] 과제:[원하는 작업] 제약:[규칙/조건] 형식:[출력 형식]

아티팩트 유형

  • 웹사이트 / 랜딩 페이지
  • 대시보드 / 리포트
